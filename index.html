The.X 運用

Twitter運用 / アルゴリズム / ストーリー戦略

この章の要点は、「Twitter運用＝フォロワー数ではなく、“人を行動させて商品を買ってもらう導線作り”」という考え方です。

全体の核心

著者は、
「良い運用＝フォロワー数やインプ数」ではなく、

読者を行動させる
ファン化する
商品購入につなげる

ことが本質だと言っています。

つまり、

興味 → 信頼 → 行動 → 購入

この流れを設計するのがTwitter運用。

3つのフェーズ
① 認知・興味フェーズ

目的：
「この人なんか気になる」と思わせる段階。

必要なのは“有益情報”ではなく、
まず圧倒的な興味づけ。

使う要素
実績
過激な意見
常識破壊
ストーリー
尖ったキャラ
理想の未来を見せる投稿

ポイント
人は最初から役立つ情報を求めていない。
まず「なんだこいつ？」と思わせることが重要。

② フォローフェーズ
目的：
信頼できる人へ進化

重要：
ノウハウ / 行動 / 挑戦 / 失敗 / 日常 / 自己投資

「背中で語る」

③ ファン化フェーズ
目的：
小さな行動 → 最終購入

いいね → リプ → RT → メルマガ → 購入

第6章まとめ

人は“情報”ではなく“感情が動いた人”に惹かれる

重要：
・理想を見せる
・敵を作る
・弱さを見せる
・挑戦を見せる
・感情を乗せる

順番：
興味 → ファン化 → 価値提供

多くの人は逆
→ 有益から入る
→ 伸びない

理由：
感情が動いていないから

ファン化：
・共感
・羨望
・悔しさ
・応援
・怒り
・期待

ノウハウはコピーされるが
キャラはコピーされない
