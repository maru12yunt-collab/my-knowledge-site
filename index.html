The.X 運用

Twitter運用 / アルゴリズム / ストーリー戦略

この章の要点は、
Twitter運用＝フォロワー数ではなく“人を行動させて商品を買ってもらう導線作り”

全体の核心：

「良い運用＝フォロワー数やインプ数」ではなく

・読者を行動させる
・ファン化する
・商品購入につなげる

つまり
興味 → 信頼 → 行動 → 購入

これが設計思想。

① 認知・興味フェーズ
目的：気になる存在になる

必要なのは有益ではなく
圧倒的な興味づけ

要素：
実績 / 常識破壊 / ストーリー / 理想 / 尖り

② フォローフェーズ
目的：信頼獲得

重要：
ノウハウ / 行動 / 挑戦 / 失敗 / 日常

背中で語る

③ ファン化フェーズ
小さな行動 → 購入

いいね → RT → リプ → 購入

2022年12月 インプ公開
2023年4月 アルゴリズム公開

目的は
Twitter社が儲かる運用を増やすため

収益：
・広告
・Twitter Blue

→ 滞在時間が最重要

ツイートスコア：

滞在時間が全て
